Edit mode: choose a section radio button and that section loops while the beat runs. Chord Beats set how long each chord lasts.

ℹ️ How to use D0GWVLK
🎵 Sections
Add Verse / Chorus / Bridge / Middle 8 / Change / Outro / Custom sections. Drag sections with ↑↓ to reorder.
🎹 Chord cards — Note ▲▼
Step the chord root up/down through the section scale notes (with octave-aware offsets preserved).
🎸 Chord cards — Bass ▲▼
In Free bass mode, adjust each chord segment's bass root independently using scale-aware steps.
🎹 Chord cards — Type ▲▼
Cycle through chord variations: maj → min → 5 → 6 … and all extended types.
🎹 Chord cards — Xpose ▲▼
Transpose only this chord up or down by one semitone.
♫ Play chord
Auditions the chord through your browser using the Web Audio API.
🔑 Section scale
Each section has its own scale root and scale type. The notes in the chosen scale are shown inline.
🔀 Global Transpose ▲▼
Shifts every chord root and every section scale root by one semitone across the whole song.
🥁 Beat
An 8-lane 16th-note drum sequencer replaces the old preset pattern. Lanes: Kick, Snare, Cl. Hat, Op. Hat, Hi Tom, Mid Tom, Low Tom, Ride. Click any step button to toggle it on/off.
🎛️ Drum Patterns (10 slots)
10 nameable drum patterns are stored per song. Use the Edit pattern dropdown in the sequencer to switch which pattern you are editing. Rename it with the Pattern name field. Patterns are saved in the browser and included in JSON export/import.
📌 Per-chord drum pattern
Each chord card has its own Drums selector plus copy/paste buttons. Choose which of the 10 patterns that chord should use, and the sequencer follows the current chord as playback advances.
🌊 Per-chord Amp LFO pattern
Each chord card has its own Amp LFO selector plus copy/paste buttons. Edit LFO patterns in the synth rack; assignments stay tempo-synced to the shared transport. Select Custom to draw a 4-beat amplitude curve, and choose Straight / Dotted / Triplet timing.
📍 Go to / position
In Song or Looping mode, use the Go to slider to jump to a song location. The position display shows current section, chord, and beat.
⏱️ Chord Beats
Each chord defaults to 4 beats (1 bar). Change Beats on each chord card to control progression length.
🎛️ Chord/Bass sounds
Chord and bass each have compact 2-oscillator synth panels with presets, waveform selectors (Osc 1/Osc 2), per-synth Transpose, ADSR, filter, harmonic drive (distortion), simple modulation (rate/depth), and tone shaping controls (Osc2 interval, detune, mix). The bass panel also includes a Pitch mode switch: Linked (bass follows chord) or Free (independent bass pitch).
↻ Chord Loop + Arp
Each chord card has a Loop toggle for Looping mode and an Arp selector (Off/Up/Down/Up and down) that changes chord-note articulation only.
🐞 Debug panel
Use the header Debug button to open a copyable runtime snapshot for transport/audio diagnostics.
🎚️ Mixer + Reverb
Use Mixer to balance chord/bass/drum/output levels. The Drums slider controls the volume of all drum sequencer lanes as well as crash and roll sounds. Drum Sequencer includes drum-machine-only FX (distortion, filter with selectable slope up to 96 dB/oct + optional LFO, and end-of-chain reverb mix/size/decay). Each synth has its own Reverb wet/dry slider, so chord and bass ambience can be shaped independently.
🔁 Note repeats
Each chord card has Chord x and Bass x selectors (1/2/4). Repeats split the same segment into repeated hits over that segment length.
🧱 DAW lanes
Each section shows chord and bass arrangement lanes. Drag blocks to reorder the progression; the chord cards update to match. Lanes keep readable minimum block widths and scroll horizontally inside the lane when needed. The currently playing block is highlighted during playback.
🧭 Arranger
Add section entries, set repeat counts, and drag to reorder. The arranger is placed below synth controls. Song mode follows arranger rows/repeats when present; otherwise it uses section order.
🪜 Octave-aware transpose
Transpose can keep moving past a single octave. Note labels still show readable pitch classes, with an extra octave/semitone offset where useful.
🥁 Section Crash / Roll
Each section has checkboxes for a crash cymbal at section start and a simple roll at section end.
💾 Save / Export / Import
Save stores songs in your browser and remembers the last opened song. Use Open most recent to load recent songs. Save Project File… / Load Project File… writes or opens a full project file from your laptop (with browser file picker fallback where needed). Export JSON / Import JSON uses the same full-project data format for portability and backwards-compatible loading. Use Export WAV / Export MIDI to render the full arranged song to audio or MIDI. 🆕 New clears everything.
