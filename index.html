ℹ️ How to use Chordz
🎵 Sections
Add Verse / Chorus / Bridge / Middle 8 / Change / Outro / Custom sections. Drag sections with ↑↓ to reorder.
🎹 Chord cards — Note ▲▼
Step the chord's root note up or down by one semitone (C → C# → D …).
🎹 Chord cards — Type ▲▼
Cycle through chord variations: maj → min → 5 → 6 … and all extended types.
🎹 Chord cards — Xpose ▲▼
Transpose only this chord up or down by one semitone.
♫ Play chord
Auditions the chord through your browser using the Web Audio API.
🔑 Section scale
Each section has its own scale root and scale type. The notes in the chosen scale are shown inline.
🔀 Global Transpose ▲▼
Shifts every chord root and every section scale root by one semitone across the whole song.
🥁 Beat
A simple kick/snare/hi-hat pattern plays at the selected BPM. Adjust BPM with + / − or type a value.
💾 Save / Export / Import
Save stores the song in your browser (localStorage). Export downloads a .json file. Import loads a saved file. 🆕 New clears everything.
