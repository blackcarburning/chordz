ℹ️ How to use Chordz
🎵 Sections
Add Verse / Chorus / Bridge / Middle 8 / Change / Outro / Custom sections. Drag sections with ↑↓ to reorder.
🎹 Chord cards — Note ▲▼
Step the chord's root note up or down by one semitone (C → C# → D …).
🎹 Chord cards — Type ▲▼
Cycle through chord variations: maj → min → 5 → 6 … and all extended types.
🎹 Chord cards — Xpose ▲▼
Transpose only this chord up or down by one semitone.
♫ Play chord
Auditions the chord through your browser using the Web Audio API.
🔑 Section scale
Each section has its own scale root and scale type. The notes in the chosen scale are shown inline.
🔀 Global Transpose ▲▼
Shifts every chord root and every section scale root by one semitone across the whole song.
🥁 Beat
A simple kick/snare/hi-hat pattern plays at the selected BPM. In Edit mode, the selected section radio loops. In Song mode, the full arrangement plays through once.
📍 Go to / position
In Song mode, use the Go to slider to jump to a song location. The position display shows current section, chord, and beat.
⏱️ Chord Beats
Each chord defaults to 4 beats (1 bar). Change Beats on each chord card to control progression length.
🎛️ Chord/Bass sounds
Chord and bass now each have a compact 2-oscillator synth panel with presets, ADSR, filter cutoff, and resonance. Bass sits directly under the chord synth controls and can be enabled independently.
🧱 DAW lanes
Each section shows chord and bass arrangement lanes. Drag blocks to reorder the progression; the chord cards update to match. The currently playing block is highlighted during playback.
🪜 Octave-aware transpose
Transpose can keep moving past a single octave. Note labels still show readable pitch classes, with an extra octave/semitone offset where useful.
🥁 Section Crash / Roll
Each section has checkboxes for a crash cymbal at section start and a simple roll at section end.
💾 Save / Export / Import
Save stores the song in your browser (localStorage). Export downloads a .json file. Import loads a saved file. 🆕 New clears everything.
